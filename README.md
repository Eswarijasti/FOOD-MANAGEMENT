# Sales Data Analysis & Dashboard (End-to-End Project)

## Problem Statement

Businesses struggle to track sales performance and identify trends across products and regions.
This project analyzes sales data to generate actionable insights for decision-making.

---

## Tech Stack

* Python (Pandas)
* MySQL
* Power BI
* Streamlit

---

## Dataset

* Records: ~10,000+ rows
* Features: Product, Region, Sales, Date, Revenue

---

## Approach

1. Data cleaning and preprocessing using Python
2. Created structured tables in MySQL
3. Wrote SQL queries using:

   * JOIN
   * GROUP BY
   * Aggregations
4. Built Power BI dashboard for KPI tracking
5. Developed Streamlit app for interactive analysis

---

## Key Insights

* Top 20% products contribute to majority of revenue
* Certain regions show consistent high sales growth
* Seasonal trends impact monthly performance

---

## Streamlit App

* Interactive filtering of sales data
* Dynamic visualization of KPIs
* Real-time query-based insights
